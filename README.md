# 지출기록부 만들기 💰

[🎬 구현화면 보기](https://laughing-tesla-bbd542.netlify.app/)

## 설치 및 시작 
```
$ npm intall 
$ npm start
```

## 사용한 기술 
- React, Context API, immer, styled-components 
- netlify

### 주요 기능 

- 기본 기능

1. 오늘의 날짜 표시
2. 현재까지 등록된 지출의 총합 표시

- CRUD

1. 지출 등록
2. 등록된 지출 삭제
3. 등록된 지출의 수정
4. 등록된 지출을 카테고리별로 모아 보기
